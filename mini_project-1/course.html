Overview

Course Code: IT4409

Course Name: Web Technologies and Online Services

Credits: 3 (2 theory - 1 practice)

This course is an important part of the IT engineering training program, helping students grasp and apply the latest technologies in building web-based software systems.

Objectives

Provide knowledge and skills for full-stack web development:

  • Understand Client-Server architecture, HTTP Protocol.
  • Master HTML5, CSS3, JavaScript.
  • Know how to use popular frameworks today.
  • Understand basic web security and design RESTful API services.

Contents

Contents include frontend programming, backend, database administration and deployment (deploy) web applications to real servers (Cloud).

IT4409 - Web Technologies

Practical project building static websites using HTML, CSS and W3.CSS framework.
